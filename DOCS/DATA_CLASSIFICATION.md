# DATA_CLASSIFICATION.md — Clasificación de datos de Aiagro.cl
> Vigente desde 2026-06-01

## Categorías activas

- **PÚBLICO:** información sin restricciones
- **INTERNO:** información de uso interno de Aiagro
- **CONFIDENCIAL:** datos de clientes, predios, métricas
- **RESTRINGIDO:** credenciales, secretos, datos personales sensibles

## Regla fundamental

Ningún dato CONFIDENCIAL o RESTRINGIDO en ambientes LOCAL o DEV.
