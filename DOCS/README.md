# Aiagro.cl — Plataforma Agrícola Inteligente

Aiagro.cl es una plataforma SaaS de gestión agrícola con inteligencia artificial,
diseñada para apoyar la toma de decisiones agronómicas en tiempo real.

## Estructura del repositorio documental

- `_MEMORY/` — Contexto de sesión para Claude Code (leer primero)
- `_BITACORA/` — Historial cronológico inmutable del proyecto
- `ADR/` — Architecture Decision Records
- `DOCS/` — Documentación viva del proyecto
- `MODULES/` — Registro de módulos del sistema
- `INFRA/` — Documentación de infraestructura cloud
- `QA/` — Estrategia y evidencia de calidad
- `SRE/` — Runbooks y procedimientos operacionales

## Cómo usar este repositorio

1. Leer `_MEMORY/PROJECT_CONTEXT.md` para entender el estado actual
2. Leer `_MEMORY/ACTIVE_TASK.md` para conocer la tarea en curso
3. Leer `_MEMORY/SESSION_HANDOFF.md` para contexto de sesión anterior
