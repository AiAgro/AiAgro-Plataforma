# SECURITY.md — Política de Seguridad de Aiagro.cl
> Reglas activas desde 2026-06-01

## Reglas críticas (vigentes desde hoy)

- No almacenar secretos en repositorio ni en archivos .md
- No exponer service role en frontend
- RLS obligatorio para todos los datos sensibles
- RBAC obligatorio
- Auditoría obligatoria en acciones críticas
- No usar datos reales en ambientes LOCAL y DEV
- Todo secreto debe gestionarse mediante variables de entorno
